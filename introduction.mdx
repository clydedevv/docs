# **Introducing maxBTC**

maxBTC is liquid, yield-bearing Bitcoin that offers sustainable BTC-denominated returns through the JLP BTC-Neutral strategy. It operates with transparent mechanics, oracle-driven exchange rate calculations, and a framework built to scale responsibly over time.

The asset uses a clear risk management framework that governs system operations and maintains exposure within set parameters.

Designed for composability, maxBTC's LST-like form factor integrates seamlessly with DeFi and can be used as reliable collateral in lending markets. Holders can deploy capital while retaining BTC exposure and liquidity.
